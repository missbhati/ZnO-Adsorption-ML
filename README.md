# ZnO-Adsorption-ML
Aim: ML model to predict adsorption energies on ZnO surfaces using DFT descriptors. \
Dataset: The dataset comprises adsorption energies of eight gas molecules (H₂S, SO₂, NO₂, NO, CO, CO₂, CH₄, and H₂) on five ZnO surfaces, resulting in a total of 40 data points. The data are derived from Density Functional Theory (DFT) calculations reported in published work.
